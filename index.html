LIGNES DE COMMANDE LINUX

  • Afficher le manuel d'une commande
    • man "..."
  • Afficher le contenu du dossier courant
    • ls
  • Afficher le contenu du dossier courant + fichiers cachés
    • ls -a
  • Afficher le contenu du dossier courant version longue
    • ls -l
  • Afficher le contenu du dossier courant version longue + fichiers cachés
    • ls -la
  • Afficher le chemin du dossier courant
    • pwd
  • Se déplacer dans un dossier
    • cd "..."
  • Se mettre en superUser
    • sudo -s
  • Mettre à jour liste des paquets
    • apt update
  • Mettre à jour les paquets
    • apt upgrade
  • Effacer l'écran
    • clear
  • Chercher tous les paquets content un mot particulier
    • apt search "..."
  • Installer un paquet en particulier
    • apt install "..."
  • Affiner une rechercher en cherchant un terme dans les résultats de "apt search"
    • apt search "..." | grep "..."
  • Revenir au dossier home
    • cd ~
  • Installer un .deb en allant dans le dossier correspondant
    • dpkg -i "..."
  • Sortir du mode superUser
    • exit
  • Créer un dossier dans le dossier courant
    • mkdir "..."
  • Créer dossier + sous-dossier
    • mkdir -p "..." / "..."
  • Créer un fichier
    • touch "..."
  • Éditer un fichier texte
    • nano "..."
  • Déplacer un fichier
    • mv "..."   "..." / "..."
