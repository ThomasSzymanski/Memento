LIGNES DE COMMANDE GIT(HUB)

  • Créer son dépôt dans le dossier courant
    • git init
  • Vérifier l'état du .git
    • git status
  • AJouter un fichier au .git
    • git add "..."
  • Créer un nouveau commit
    • git commit -m "..."
  • Créer branche pour modifier sans toucher au projet
    • git branch "..."
  • Changer de branche
    • git checkout "..."
  • Fusionner branche master + autre branche
    • git merge "..."
  • Supprimer une branche
    • git branch -d "..."
  • Envoyer le fichier du PC à GitHub
    • git push origin
  • Envoyer le fichier de GitHub au PC
    • git pull origin
